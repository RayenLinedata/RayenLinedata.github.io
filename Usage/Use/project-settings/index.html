    Steps to configure your project

    First step : Configuration (important)

    In the first step of project configuration you must enter :

    methodology : SCRUM, CR or WaterFall.

    client name : name of the client.

    team members: choose the members who will work on the project.

    Second step : Estimation (important)

    Estimation time is the periode of time on (days) that every project will take to be done.

    Third step : Risks (optional)

    Fourth step : Assumptions (optional)

    Last step

    Finally click on Add this project and your project will be added to setted project on the left side and you will be redirected to projects page View projects .
